# Source Priority

When metadata conflicts, prefer sources in this order:

1. Publisher or official venue page
2. DOI landing page / Crossref-style metadata
3. Official proceedings page
4. Structured scholarly indexes:
   - DBLP
   - OpenAlex
   - Semantic Scholar
   - OpenReview
   - arXiv
5. Institutional repository
6. Aggregators, blogs, mirror pages

## Practical rule

If a publisher page and an aggregator disagree, prefer the publisher page unless it is obviously incomplete.

## DOI rule

If a DOI resolves to a formal publisher page, that usually outranks a secondary source summary.

## Preprint rule

arXiv is authoritative for the existence of the preprint version, but not for proving formal publication.

# Publication Type Rules

Use these rules to classify a paper conservatively.

## Journal

Strong signals:

- Official journal article page on a publisher site
- Journal volume / issue / page information
- DOI metadata clearly naming a journal
- Article record on the journal's official site

## Conference

Strong signals:

- Official proceedings page
- ACM / IEEE / Springer proceedings record
- DBLP or publisher metadata clearly naming a conference proceeding
- DOI metadata naming proceedings or conference publication

## Preprint

Strong signals:

- arXiv
- SSRN
- bioRxiv
- medRxiv
- OpenReview preprint-style record without confirmed proceedings publication

## Important distinctions

- A paper can exist both as a preprint and as a formal publication.
- If both exist, do not collapse them into one label silently.
- Report the relationship explicitly:
  - preprint version exists
  - formal journal publication exists
  - formal conference publication exists

## If unsure

Return `unknown` or `conflicting`, with notes.

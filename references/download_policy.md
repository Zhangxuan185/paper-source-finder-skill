# Download Policy

This skill should only download PDFs that are directly and lawfully accessible.

## Allowed

- Open-access publisher PDFs
- arXiv PDFs
- Institutional repository PDFs
- Publicly accessible author copies when the source is clearly legitimate

## Not allowed

- Bypassing paywalls
- Using leaked mirrors
- Using shadow libraries
- Circumventing login restrictions without explicit lawful access

## Workflow

1. Check whether a direct PDF URL exists.
2. Confirm it returns a PDF-like content type or a clear PDF file.
3. Save it to the requested folder.
4. Report the local file path.

## If download is not allowed

Return:

- source page URL
- PDF URL if visible but inaccessible
- a short note explaining that download was skipped for access/compliance reasons

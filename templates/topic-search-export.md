# Topic Search + Export Template

```text
Use paper-source-finder for this task.

请按主题帮我搜索论文：
【主题】

要求：
1. 先找出相关论文
2. 判断每篇是期刊、会议还是预印本
3. 给出 venue 名称
4. 给出 venue 等级和所用 ranking framework
5. 返回字段：
   - 标题
   - 作者
   - 年份
   - 类型
   - venue
   - ranking
   - ranking framework
   - DOI
   - source URL
   - PDF URL
6. 最后导出成【Markdown 表格 / CSV / JSON】
7. 如果有可直接合法访问的 PDF，并且我说需要下载，也请下载到【目标目录】
```
